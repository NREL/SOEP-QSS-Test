FloorOpenLoop Model Notes

This is a building thermal model that is a good validation case for the
QSS solver.

This model is currently set up for Linux: it won't build on Windows.

This model is contained in an issue branch of the Buildings Library:
git clone -b issue1129_energyPlus_zone git@github.com:lbl-srg/modelica-buildings.git
It was Buildings.Experimental.EnergyPlus.Examples.VAVReheatRefBldgSmallOffice.FloorOpenLoop
Then moved to Buildings.ThermalZones.EnergyPlus.Examples.VAVReheatRefBldgSmallOffice.FloorOpenLoop
